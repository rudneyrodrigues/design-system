# Button

O componente de `Button` é utilizado para interações do usuário. Ele pode ser utilizado para ações como salvar, enviar, deletar, etc.

<Canvas of={ButtonStories.Primary} />

## Variantes

O `Button` possui 4 variantes: `primary`, `secondary`, `ghost` e `outline`.

### Primary

O botão `primary` é utilizado para ações principais, como salvar, enviar, etc.

<Canvas of={ButtonStories.Primary} />

### Secondary

O botão `secondary` é utilizado para ações secundárias, como cancelar, voltar, etc.
